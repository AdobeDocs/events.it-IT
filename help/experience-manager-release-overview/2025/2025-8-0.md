# Video introduttivo sulla versione 2025.08.0

Le funzioni di questa versione si concentrano sulle innovazioni di Experience Manager Sites, Forms, Assets e Cloud Foundation e includono:

## Cloud Foundation

* Migliora la sicurezza e la fiducia dei clienti fornendo una connessione sicura e l’automazione alla CMK di AEM&#x200B;
* Produce OIDC per la pubblicazione - Jira
* Porta il tuo Git (BYOG)

## Moduli

* Efficienza dell’authoring grazie alla creazione più semplice delle regole
* Pulsante preimpostato per il download del documento record

## Siti

* Contrassegna i tuoi contenuti
* Informativa sulla sicurezza dei contenuti
* Porta il tuo Git
* Snapshot e revisione
