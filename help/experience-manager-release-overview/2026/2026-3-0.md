# Video introduttivo sulla versione 2026.03.0 di AEM

>[!VIDEO](https://video.tv.adobe.com/v/3483060/?learn=on&enablevpops)

Le funzioni di questa versione si concentrano sulle innovazioni di Experience Manager Sites, Forms, Assets e Cloud Foundation e includono:

* Siti
   * Supporto di aggiornamenti e avvii di Experience Production Agent

* Moduli
   * Supporto del flusso di lavoro per l&#39;editor IC
   * Supporto dell’Editor temi per i componenti core
   * Supporto del controllo delle versioni per Forms

* Risorse
   * AEM Content Advisor - Accesso GA con le app Adobe
   * Connettore Microsoft Office (PowerPoint, Word, Excel)

* Cloud Foundation
   * Agente di governance: gestione del marchio
   * Autorizzazioni di controllo sull’interfaccia utente touch
   * Funzionalità di disconnessione OIDC
   * Sospendi aggiornamenti di manutenzione

* Cloud Manager
   * Server Cloud Manager MCP
