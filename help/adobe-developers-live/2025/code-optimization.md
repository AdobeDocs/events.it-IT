# All’interno del motore: ottimizzazione del codice con Adobe Experience Manager Sites Optimizer

Scopri in che modo Adobe Experience Manager Sites Optimizer aiuta gli sviluppatori a migliorare la qualità del codice rilevando script ridondanti e strutture DOM pesanti. Questa sessione illustra come utilizzare lo strumento per migliorare Core Web Vitals, ridurre il debito tecnico e migliorare le prestazioni. Scopri come applicare i consigli direttamente nel flusso di lavoro, per correzioni più rapide, SEO (Search Engine Optimization) più solidi e esperienze utente più fluide.

>[!VIDEO](https://video.tv.adobe.com/v/3477676/?learn=on&enablevpops)

Questa sessione - Inside the Engine: Code Optimization with Adobe Experience Manager Sites Optimizer - presenta Deck Reyes che illustra come Sites Optimizer analizza le implementazioni front-end e back-end per rilevare script ridondanti, strutture DOM pesanti e colli di bottiglia di Core Web Vitals. Scopri come applicare le raccomandazioni per ridurre il debito tecnico e migliorare le prestazioni direttamente in AEM. Registrato dal vivo da San Jose.
