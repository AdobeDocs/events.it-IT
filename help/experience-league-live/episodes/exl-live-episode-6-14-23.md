# Esplora il futuro di Adobe Journey Optimizer con il canale web e la sperimentazione dei contenuti

Il nuovo canale web consente agli addetti al marketing di creare campagne web con un designer visivo di WYSIWYG. Inoltre, Journey Optimizer ora consente di sperimentare contenuti in qualsiasi canale: e-mail, web, push, in-app, SMS e altro ancora.
