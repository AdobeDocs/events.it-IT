# Approfondimento su AEM 5.6 Media Publisher {#aem-media-publisher-deep-dive}

Come parte di Adobe Experience Manager, CQ 5.6 fornisce un nuovo framework Commerce per creare siti web Experience Driven Commerce sopra una piattaforma di e-commerce di terze parti. Questa sessione fornisce una panoramica del framework da un punto di vista architettonico e presenta alcuni dettagli dell’implementazione di riferimento, basata sull’archivio JCR.
