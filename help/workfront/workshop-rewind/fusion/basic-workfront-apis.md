# Nathan&#39;s Five Faves - API di base per il mondo Workfront

Sei curioso di scoprire cosa può fare veramente l’API di Workfront per te? Unisciti a Nathan Johnson, ingegnere del supporto tecnico Adobe III, che analizza le nozioni di base e porta le API nel mondo reale.

In questa sessione, Nathan esamina le nozioni di base dell’API di Workfront, cos’è, come funziona e perché è importante, prima di immergersi nei suoi cinque casi d’uso preferiti che ne mostrano il potere in azione.

>[!VIDEO](https://video.tv.adobe.com/v/3492587/?learn=on&enablevpops)

Dalle conoscenze fondamentali alle applicazioni nel mondo reale, questa sessione ti offrirà un percorso chiaro e accessibile nel mondo delle API Workfront, guidato da un esperto che lavora con loro ogni giorno. Che tu sia un nuovo utente delle API o desideri espandere il tuo toolkit tecnico, questa sessione ti aiuterà a collegare i punti tra concetto e funzionalità.
